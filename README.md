# Expense tracker

Track your expenses in from your terminal and store them on an sqlite file

## Installation

Navigate to the cloned directory and run

```bash
   ❯ source install.sh
```

## Usage

```bash
   ❯ expenses
```

Output:

```bash
	Add expense
	View expenses
	Edit expense
	Remove expense
	Quit	

```

The  db.sqlite  file will be created in them `/home/[user]/expenses` directory
